# Kilimbe Microfinance

Welcome to the Kilimbe Microfinance application repository!

## Getting Started

To run the application locally:

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Building for Production

```sh
npm run build
```
